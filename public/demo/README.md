# TEXNOVA 플랫폼 스크린샷 가이드

이 폴더에 TEXNOVA 플랫폼의 실제 구동 화면 스크린샷을 추가하세요.

## 필요한 이미지 파일

다음 파일명으로 스크린샷을 저장해주세요:

1. **dashboard.png** - 대시보드 화면
   - 실시간 모니터링 대시보드
   - 데이터 수집 현황, 분석 진행 상태 등

2. **analysis.png** - 텍스트 분석 화면
   - 토픽모델링 결과
   - 감성분석 결과
   - 키워드 추출 화면

3. **network.png** - 네트워크 분석 화면
   - 동시출현 네트워크 그래프
   - 중심성 분석 결과

4. **visualization.png** - 시각화 화면
   - 워드클라우드
   - 차트 및 그래프
   - 리포트 화면

## 이미지 권장 사양

- **형식**: PNG (투명 배경 가능)
- **크기**: 1920x1080px (16:9 비율)
- **용량**: 각 파일 2MB 이하
- **품질**: 고해상도 (Retina 디스플레이 대응)

## 추가 방법

1. 위 파일명으로 스크린샷을 저장
2. 이 폴더(`public/demo/`)에 파일 복사
3. 웹사이트에서 자동으로 표시됩니다

## 주의사항

- 실제 고객 데이터가 포함되지 않도록 주의
- 샘플 데이터나 테스트 데이터 사용 권장
- 민감한 정보는 모자이크 처리
