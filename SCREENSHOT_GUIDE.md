# TEXNOVA 플랫폼 스크린샷 캡처 가이드

## 📸 스크린샷 캡처 방법

### 1. TEXNOVA 플랫폼 접속

TEXNOVA 백업 시스템이 실행 중이라면 다음 URL로 접속하세요:
- **프론트엔드**: http://localhost:3000
- **백엔드**: http://localhost:7000

### 2. 필요한 스크린샷 (4개)

#### ① dashboard.png - 대시보드 화면
**캡처할 화면**:
- 메인 대시보드 페이지
- 파일 업로드 인터페이스
- 분석 옵션 선택 화면
- 프로젝트 목록 (있다면)

**캡처 방법**:
1. 메인 페이지 접속
2. Windows 키 + Shift + S (화면 캡처 도구)
3. 전체 화면 또는 중요 영역만 선택
4. 저장: `dashboard.png`

#### ② analysis.png - 텍스트 분석 화면
**캡처할 화면**:
- 텍스트 분석 결과 페이지
- 토픽 모델링 결과
- 감성 분석 결과
- 키워드 추출 결과

**샘플 분석 실행하기**:
1. 테스트용 텍스트 파일 업로드
2. "단어 빈도 분석" 또는 "토픽 모델링" 선택
3. 분석 실행
4. 결과 화면 캡처
5. 저장: `analysis.png`

#### ③ network.png - 네트워크 분석 화면
**캡처할 화면**:
- 네트워크 그래프 시각화
- 동시출현 분석 결과
- 중심성 분석 결과

**샘플 분석 실행하기**:
1. 텍스트 파일 업로드
2. "네트워크 분석" 선택
3. 분석 실행
4. 네트워크 그래프가 표시되면 캡처
5. 저장: `network.png`

#### ④ visualization.png - 시각화 화면
**캡처할 화면**:
- 워드클라우드
- 막대 그래프 / 선 그래프
- 시계열 분석 차트
- 리포트 화면

**샘플 분석 실행하기**:
1. 텍스트 파일 업로드
2. "단어 빈도 분석" 선택
3. "워드클라우드 생성" 옵션 선택
4. 분석 실행
5. 워드클라우드가 표시되면 캡처
6. 저장: `visualization.png`

### 3. 스크린샷 편집 (선택사항)

### 5. 빠른 스크린샷 방법

#### Windows 스크린샷 단축키
- **Win + Shift + S**: 화면 일부 캡처 (권장)
- **Win + PrtScn**: 전체 화면 캡처
- **Alt + PrtScn**: 현재 창만 캡처

#### 캡처한 이미지 저장
1. 클립보드에 복사된 이미지를 그림판 열기 (mspaint)
2. Ctrl + V로 붙여넣기
3. "다른 이름으로 저장" → PNG 형식 선택
4. 파일명 지정 (dashboard.png 등)
5. public/demo/ 폴더에 저장
